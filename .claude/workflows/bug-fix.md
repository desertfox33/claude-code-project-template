# Bug Fix Workflow

Use this workflow for defect investigation and repair.

## Steps

1. Reproduce or understand the failure.
2. Identify the smallest affected area.
3. Confirm likely root cause.
4. Make the smallest safe fix.
5. Add regression coverage.
6. Run the narrow test first.
7. Summarize root cause and validation.

## Output

- Root cause
- Fix summary
- Regression test
- Commands run
- Remaining risk
